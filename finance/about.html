About This Project

This website provides tools to calculate key financial values such as Future Value (FV) and Present Value (PV).

Purpose

The goal of this project is to help students, investors, and learners better understand the time value of money. By using these calculators, you can easily explore how interest, growth, and deposits affect your financial outcomes.

How It Works

Each calculator uses standard finance formulas, implemented with clear JavaScript logic. You enter the values, and the app handles the computations instantly—no dependencies, no backend, just clean and fast client-side math.

What’s Included

  • Future Value (FV) Calculator
  • Present Value (PV) Calculator
  • Simple, clean user interface
  • Accurate math formulas implemented in vanilla JavaScript

Author

Created as a learning project to practice financial concepts, HTML/CSS layout, and JavaScript functionality — while maintaining a minimal aesthetic using JetBrains Mono and a structured container-based layout.
